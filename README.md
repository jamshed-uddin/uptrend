## upTrend

### Run the app locally

### 1. Clone the repository

```bash
git clone https://github.com/jamshed-uddin/uptrend.git

cd uptrend
```

### 2. Install the packages

```bash
npm install
```

### 3. Start the app

```bash
npm run dev
```
